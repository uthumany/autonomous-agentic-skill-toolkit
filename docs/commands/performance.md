# Performance Testing

## `aast test:performance <url>`

Runs a performance audit on a specified URL using Lighthouse. This command provides insights into the web application's performance, accessibility, best practices, SEO, and Progressive Web App (PWA) metrics.

### Arguments

*   `<url>` (required): The URL of the web application to audit for performance.

### Examples

```bash
aast test:performance https://example.com
```
