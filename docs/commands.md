# Commands

This section provides a detailed reference for all commands available in the Autonomous Agentic Skill Toolkit CLI (`aast`).

## Overview

The `aast` CLI is designed to be run from your terminal, providing a suite of tools for automated testing across various platforms. Each command is self-contained and can be executed independently or as part of a larger automation script.

## Command Structure

All commands follow a similar structure:

```bash
aast <command> [arguments] [options]
```

*   `<command>`: The specific action to perform (e.g., `test:web`, `generate:report`).
*   `[arguments]`: Required parameters for the command (e.g., `<url>`, `<testResultsFile>`).
*   `[options]`: Optional flags to modify command behavior (e.g., `--device`, `--format`).
